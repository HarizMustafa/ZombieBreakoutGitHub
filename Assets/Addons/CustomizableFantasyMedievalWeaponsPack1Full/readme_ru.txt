1. Содержимое пакета
--------------------
В пакет входят 20 предметов фэнтезийного вооружения:
* двуручный меч;
* полуторный меч;
* одноручный меч;
* кинжал;
* одноручный топор;
* двуручный топор;
* булава;
* одноручный молот;
* двуручный молот;
* алебарда;
* копье;
* боевой посох;
* магический посох;
* колчан;
* стрела;
* арбалет;
* лук;
* малый щит;
* средний щит;
* большой щит.

Все модели содержат по 2 lod'a, каждый из которых урезается по полигонажу в 2 раза. Исключение составляют модели стрелы и одноручного меча.

Помимо них пакет содержит преднастроенные демонстрационные 4 версии префабов для каждого предмета и демонстрационную сцену.

Также он содержит демонстрационный шейдер. Внимание, шейдер создавался исключительно для демонстрации возможностей данного пакета - он не тестировался в финальных проектах. Если Вы захотите использовать полную функциональность пакета - Вам придется написать свой шейдер. Либо, как и нам, создать его из графов с помощью StrumpyShaderEditor.

Помимо этого, пакет содержит текстуры развертки для удобства редактирования текстур в случае надобности.


2. Функционал
-------------
Шейдер оперирует следующими текстурами:
* diffuse (текстуры без постфиксов, например, arrow.tga) - альфа-канал отвечает за прозрачность;
* normalmap (текстуры с постфиксом _n, например, arrow_n.tga) - карта бампа;
* specular (текстуры с постфиксом _s, например, arrow_s.tga) - карта бликов;
* glow (текстуры с постфиксом _g, например, arrow_g.tga) - карта свечения;
* reflection (текстуры с постфиксом _r, например, arrow_r.tga) - карта отражений;
* cubemap - входящая в пакет текстура псевдоотражений;
* colorise (текстуры с постфиксом _c, например, arrow_c.tga) - карта покраски.

Помимо этого есть дополнительные настройки цветов:
* diffuse - цвет диффуза;
* specular - цвет спекуляра;
* glow - цвет свечения;
* colorise для 4х каналов - цвета покраски для масок, расположенных на RGBA-каналах текстуры покраски. Красится все области на канале текстуры, отличимые от черного. На модель цвет накладывается мультиплаем.

Также есть настройки:
* glossiness - ширины луча спекуляра;
* glowness - силы свечения;
* reflectioness - интенсивности отражений.
